# React学習　犬画像ギャラリー

https://zenn.dev/likr/articles/6be53ca64f29aa035f07
こちらを参考に、TypeScriptとTailwindを導入してReactの学習。

## 技術スタック

- **Frontend**: React 19.1.1
- **Language**: TypeScript
- **Build Tool**: Vite 7.1.3
- **Styling**: Tailwind CSS 4.1.12
- **Package Manager**: Yarn 4.9.3
- **Node.js**: 24.6.0

## セットアップ

### 前提条件

- Node.js 24.6.0以上
- Yarn 4.9.3以上

### コマンド

```bash
# 依存関係をインストール
yarn install

# 開発サーバーを起動
yarn dev

# 本番用ビルド
yarn build

# ビルド結果のプレビュー
yarn preview
```

### dev起動後URL
http://localhost:5173
